## 📰 News App

A modern, high-fidelity news application built with Flutter.

![Flutter Version](https://img.shields.io/badge/Flutter-3.x-blue.svg)

---

## ✨ Overview

News App is a powerful and elegant news aggregator designed to provide users with a seamless reading experience. It combines modern UI design with efficient data handling to deliver curated news content based on user preferences.

The app is built with scalability in mind and follows clean architecture principles.

---

## 🚀 Key Features

* 🎯 **Personalized News Feed**
  Get news based on selected categories (sports, politics, tech, etc.)

* 🔥 **Top News Section**
  Stay updated with trending global headlines

* 🔐 **Authentication**

  * Email & Password
  * Google Sign-In (Firebase)

* 🎨 **Modern UI/UX**

  * Glassmorphism design
  * Smooth animations
  * Custom navigation bar

* 🌙 **Theme Support**

  * Light / Dark / System mode

* ⚡ **Performance Optimized**

  * Efficient state management (Riverpod)
  * Smooth scrolling experience

---

## 📱 App Structure

The app consists of **3 main screens**:

1. **Top News** – Global trending headlines
2. **Your News** – Personalized feed based on user-selected categories
3. **Profile** – User settings and account management

## ⚙️ Getting Started

```bash
# Clone the repo
git clone https://github.com/Santhosh7474/News_App.git

# Navigate into the project
cd News_App

# Get dependencies
flutter pub get

# Configure Firebase (Required)
flutterfire configure

# Run the app
flutter run
```
